
What's here
===========
  The official Eco Capture Series Card Driver for Linux

Compatible Linux Kernel Version
===============================
  Linux kernel 2.6.35 and above
  

Compatible Device
=================
  Eco Capture Dual HDMI M.2
  Eco Capture Dual SDI M.2
  Eco Capture HDMI 4K M.2
  Eco Capture Quad SDI M.2

Compatible Device Firmware Version
==================================
  Eco Capture 1.3 and above

Compiler info
=============
  GNU GCC 4.5.3 and above
